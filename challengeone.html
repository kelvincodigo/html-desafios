Source Code:
Is generally understood to mean programming statements that are created by a programmer with a text editor ou a visual programming tool and then saved in a file.
Object Code:
Generally refers to the output, a compiled file, which is produced when the Source Code is compiled with a C compiler. The object code file contains a sequence of machine-readable instructions that is processed by the CPU in a computer. Operating system or application software is usually in the form of compiled object code.
